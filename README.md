This dashboard is built for job seekers and career switchers to simplify insights from the data job market. Using 2024 data science job postings (titles, salaries, locations), it provides a clear, interactive view of trends and compensation in one place.

Key Skills Demonstrated
Data Transformation (Power Query): Cleaned and prepared raw data
Metrics & KPIs: Median salary, job count, and key insights
Visualizations: Bar, column, line, area, and map charts
Dashboard Design: Clean, user-friendly layout
Interactivity: Slicers, drill-through, buttons, and bookmarks


Dashboard Overview
Page 1 – Market Summary: High-level view of job counts, salaries, and top roles
Page 2 – Detailed Analysis: Drill-through by job title with salary ranges, remote work trends, hiring platforms, and global distribution

Outcome
Transforms raw job data into an interactive career tool to explore trends, compare roles, and make informed decisions.
